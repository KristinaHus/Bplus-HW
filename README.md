CRIOS

DEPENDENSES

To make this template work correct, you'll have to install next libraries and components:
 * Bottstrap
 * Flexboxgrid
 * jQuery
 * Gulp

QUICK START
 
To make this working you can type in your terminal
 <pre>
    <code>
        npm install
        bower install
    </code>
 </pre>
 
And after that you'll have to start server with next commands
 <pre>
     <code>
         gulp
         gulp serve
     </code>
  </pre>
